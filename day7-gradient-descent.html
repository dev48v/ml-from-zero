It's a three-line loop. The same loop trains linear regression, neural nets, everything.

    You have a loss as a function of a parameter
    const loss = (x) => (x - 5) ** 2;   // a bowl with its minimum at x = 5

  2. 2
    The gradient = the slope at your point
    const grad = (x) => 2 * (x - 5);    // derivative of the loss
// positive slope → you're on the right wall → move left, and vice-versa
    3. 
      
  3. 3
    Step DOWNHILL: subtract lr × gradient
    x = x - learningRate * grad(x);   // the whole algorithm

  4. 4
    Repeat until the slope is ~0
    for (let i = 0; i < 1000; i++) {
  x -= lr * grad(x);
  if (Math.abs(grad(x)) < 1e-4) break;   // converged at the minimum
}
