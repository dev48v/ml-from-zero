Four counts give you everything. ~12 lines.

    
    
      
  1. 1
    Count the four outcomes
    let TP=0,FP=0,FN=0,TN=0;
for (const x of data) {
  const pred = x.score >= threshold;
  if (pred && x.actual)      TP++;
  else if (pred && !x.actual) FP++;
  else if (!pred && x.actual) FN++;
  else                        TN++;
}

  2. 2
    Precision: of what I flagged, how much was right?
    const precision = TP / (TP + FP);   // few false alarms = high
    3. 
      
  3. 3
    Recall: of all the real ones, how many did I catch?
    const recall = TP / (TP + FN);   // few misses = high
    4. 
      
  4. 4
    F1: the harmonic mean (punishes imbalance)
    const f1 = 2 * precision * recall / (precision + recall);
// harmonic, not average — so 0.9/0.1 gives ~0.18, not 0.5

    Accuracy — and why it lies
    const accuracy = (TP + TN) / data.length;
// 99% of email is ham → a model that says "ham" always = 99% accuracy,
// but precision/recall on spam = 0. Always report all four.
