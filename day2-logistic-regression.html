It's Day 1's linear regression + a sigmoid. ~40 lines.

    
    
      
  1. 1
    The sigmoid squashes to 0–1
    const sigmoid = z => 1 / (1 + Math.exp(-z));
const predict = x => sigmoid(w1*x[0] + w2*x[1] + b);  // probability
    2. 
      
  2. 2
    Gradient is identical to linear's
    const err = predict(x) - y;     // y is 0 or 1
dw1 += err * x[0]; dw2 += err * x[1]; db += err;   // beautifully, same form
    3. 
      
  3. 3
    Step downhill, repeat
    w1 -= lr*dw1/n; w2 -= lr*dw2/n; b -= lr*db/n;

  4. 4
    Classify at 0.5
    const label = predict(x) >= 0.5 ? 1 : 0;
