Count words per class, apply Bayes, multiply in log-space. ~35 lines.

    
    
      
  1. 1
    Count words in each class
    for (const {text, label} of trainingData)
  for (const w of tokenize(text))
    counts[label][w] = (counts[label][w] || 0) + 1;
    2. 
      
  2. 2
    P(word | class), with +1 smoothing
    function pWordGiven(w, label){
  return (counts[label][w] || 0 + 1) / (totalWords[label] + vocabSize);
}   // +1 so an unseen word never zeroes the whole product

  3. 3
    Bayes: multiply the priors × likelihoods
    // P(spam | words) ∝ P(spam) · Π P(word | spam)
score[label] = Math.log(prior[label]);
for (const w of words) score[label] += Math.log(pWordGiven(w, label));

    // multiplying 30 probabilities → 1e-40, rounds to 0.
